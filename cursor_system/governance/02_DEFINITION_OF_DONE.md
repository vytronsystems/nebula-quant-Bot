# Definition of Done

A phase is only done when all of the following are true:

1. Code compiles/runs.
2. Relevant tests pass.
3. Existing passing tests do not regress.
4. Documentation is updated.
5. Artifact files are generated.
6. DB/file evidence is written.
7. Security impact has been checked.
8. Packaging/import paths remain valid from repo root.
9. Any new API contract is documented.
10. Any UI change is wired to real data or well-defined mock contract explicitly tagged as temporary and non-production.
