# NEBULA-QUANT v1 | Weekly Audit Standard

## Objetivo
Revisar todas las operaciones y todos los experimentos de la semana para mejorar el sistema.

## Frecuencia
Semanal

## Revisar siempre
- entradas
- salidas
- TP
- SL
- RR esperado
- RR real
- slippage
- setups ganadores
- setups perdedores
- condiciones de mercado
- errores operativos
- edge decay

## Resultado esperado
El weekly audit debe producir:
- hallazgos
- cambios sugeridos
- estrategias a ajustar
- estrategias a pausar
- nuevas hipótesis a investigar

## Regla
No se mejora el sistema solo por intuición.
Toda mejora debe quedar documentada y volver a validarse.
